# Ascii Converter

This code transforms a jpeg image and convert it to a ASCII art in the terminal, using a gray scale as it's base.

## Getting Started

### Prerequisites

To run this project you will need PIL and Python 3+  installed in your machine

```
pip3 install Pillow
```
